# Deep Learning Projects for Image Detection and Classification

A collection of computer vision and deep learning projects covering image classification, object detection, and remote sensing applications.

## Included Projects

- **CNN-based Cat vs Dog Classification**
- **Detectron2-based Chess Detection**
- **Satellite Image Classification**
- **YOLOv4 / Tiny YOLO for American Sign Language Detection**
- **Custom YOLOv5 Model**
- **Detectron2-based American Sign Language Detection**

## Topics Covered

- Image classification
- Object detection
- Deep learning
- Computer vision
- Remote sensing
- Sign language detection

## Tools and Frameworks

- Python
- Jupyter Notebook
- CNN
- Detectron2
- YOLOv4
- YOLOv5

## Repository Structure

Each folder contains a separate project implementation, dataset preparation steps, and model training or testing notebooks/scripts.

## Purpose

This repository presents different deep learning approaches for solving real-world image detection and classification tasks, and serves as a portfolio of applied computer vision projects.
