# Deep Learning Projects for Image Detection and Classification

A curated collection of deep learning and computer vision projects focused on **image classification**, **object detection**, and **applied visual AI**.

This repository brings together multiple experiments and implementations using different frameworks and model families, including **CNNs**, **Detectron2**, **YOLOv4**, and **YOLOv5**.

## Overview

The goal of this repository is to showcase practical implementations of deep learning algorithms for solving real-world image understanding tasks such as:

- image classification
- object detection
- sign language detection
- satellite image analysis
- custom model experimentation

## Repository Structure

Each folder in this repository contains a separate project, experiment, or implementation.  
The projects may include training notebooks, inference notebooks, dataset preparation steps, and model-specific experimentation files.

## Purpose

This repository is intended to present a range of applied deep learning techniques for image-based tasks and serve as a portfolio of practical computer vision work.

It highlights experimentation across different detection and classification frameworks rather than a single end-to-end system.
